## 제목

uvws – ComfyUI·파이썬 로컬 앱을 데스크탑에서 클릭 한 번으로 실행하는 런처 (uv 기반, 오픈소스)

## 본문 (내용 칸에 붙여넣기)

로컬에서 ComfyUI나 WebUI, 파이썬 스크립트 여러 개를 띄워 쓰다 보니 매번 터미널 열고 cd 하고 venv 켜고 python 돌리는 게 은근 일이라 직접 만들어 봤습니다.

ComfyUI 하나 띄우고 다른 폴더에서 또 뭐 돌리고 하면 터미널 탭이 금방 지저분해지고, 포트 충돌 나면 lsof 쳐서 kill 하는 것도 매번 반복이더라고요.

- 폴더 등록해두면 그다음부턴 클릭 한 번으로 실행 (내부적으로 `uv run`으로 감싸 venv 없어도 돌아가고, 없으면 첫 실행 때 자동 생성. **uv가 안 깔려 있으면 앱이 설치까지 해줌**)
- 각 프로젝트 로그가 앱 안 터미널(xterm.js)로 실시간으로 뜨고, `localhost:포트` 나오면 브라우저 열기 버튼이 자동으로 생김
- 포트 강제 종료 버튼 — 그 포트 잡은 프로세스 SIGKILL 때리고 진짜 비워질 때까지 확인 (제일 많이 씀)
- 여러 프로젝트 동시에 띄울 수 있고, 앱 끄면 돌던 서버들 같이 정리
- **공유 + QR** — 실행 중인 로컬 서버를 공개 링크 + QR로 한 번에 노출. 배포 없이 폰으로 QR 찍어 바로 접속 (cloudflared)
- 프로젝트별 CPU/메모리 실시간 모니터, 서버 준비/비정상 종료 시 네이티브 알림, 구버전 패키지 업그레이드(의존성 닥터)
- requirements.txt / pyproject.toml 있으면 파이썬 버전 골라 의존성 설치까지 한 번에

Tauri 2 + React로 만들었고 macOS(Apple Silicon/Intel), Windows 빌드 올려놨습니다.

아직 공증은 안 받아서 맥 첫 실행 때 보안 경고가 뜨는데 README에 우회 방법 적어놨습니다. 개인 사이드 프로젝트라 부족한 점 많을 텐데 피드백/이슈 환영합니다.
