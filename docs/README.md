# uvws 🚀
### 로컬 AI 및 Python 프로젝트를 위한 초고속 원클릭 제어 허브

> **"더 이상 반복적인 터미널 명령어와 가상환경 실행에 시간을 낭비하지 마세요."**
> 
> ComfyUI, Gradio, ACE-Step, Streamlit 등 수많은 로컬 AI 웹 서비스와 파이썬 스크립트들을 마우스 클릭 한 번으로 구동하고, 스마트하게 포트를 감지해 브라우저를 열어주는 초경량 데스크톱 오케스트레이터 **uvws**를 소개합니다.

---

## 💡 개발자의 귀찮음에서 시작된 혁신

새로운 AI 모델이나 로컬 스크립트를 실행할 때마다 매번 겪어야 했던 피로한 반복 작업들을 기억하시나요?
1. 프로젝트 폴더를 터미널로 찾아 들어가기
2. `.venv` 또는 `conda` 가상환경 활성화하기
3. `requirements.txt`나 `pyproject.toml`을 보고 누락된 의존성 설치하기
4. `python main.py` 혹은 복잡한 실행 옵션(Arguments) 입력하기
5. 서버가 뜬 후 터미널 로그를 뒤져 포트 번호를 복사하고, 브라우저 주소창에 쳐서 접속하기

**uvws는 이 모든 과정을 단 한 번의 클릭으로 압축합니다.**

---

## ✨ uvws의 핵심 가치 & 주요 기능

### 1. ⚡ Astral `uv` 결합을 통한 압도적인 의존성 속도
느리고 무거운 기존 패키지 툴 대신 가장 현대적이고 빠른 Python 패키지 매니저인 `uv`를 완벽하게 활용합니다. 가상환경 구축부터 의존성 다운로드까지 눈 깜짝할 사이에 격리 완료됩니다.

### 2. ⏱️ 1초 미만 즉시 구동 (Venv Cache 최적화)
이미 불러온 폴더에 가상환경(`.venv`)과 패키지 세팅이 완료되어 있다면, 불필요한 의존성 검사를 스킵하고 즉각적으로 백엔드 프로세스를 구동합니다. 클릭 즉시 서버가 실행됩니다.

### 3. 📂 명령어는 이제 그만, 단정한 UI와 네이티브 폴더 브라우저
경로를 일일이 타이핑할 필요가 없습니다. 네이티브 디렉토리 브라우저 창(`Browse...`)을 띄워 원클릭으로 로컬 폴더를 추가하고 관리하세요. 폴더명에 맞춰 프로젝트 이름도 자동 완성됩니다.

### 4. 🌐 스마트 포트 감지 및 원클릭 브라우저 연동
파이썬 백엔드가 구동되며 콘솔에 출력하는 서버 포트(Gradio, Streamlit, Flask, ComfyUI 등)를 정밀 분석하여, 포트 접속 버튼을 Cyan 그라데이션 컬러로 즉시 활성화합니다. 주소창을 타이핑할 필요가 전혀 없습니다.

### 5. 💻 단정한 macOS 스타일 가상 터미널 제공
프로세스가 정상 작동 중인지, 에러가 발생하진 않는지 확인하기 위해 별도의 터미널을 열 필요가 없습니다. 삼색 윈도우 컨트롤러가 포함된 단정한 macOS 가상 터미널 안에서 실시간 `stdout`/`stderr`(빨간색 에러 하이라이트) 로그를 확인하세요.

## 🎨 군더더기 없는 미니멀리스트 라이트 디자인

uvws는 화려하고 번잡한 스타일을 지양합니다. 
눈의 피로를 최소화하는 **소프트 화이트 및 징크 라이트 그레이 톤**의 정돈된 레이아웃 안에서, 개발 생산성을 높일 수 있는 에센셜 기능들만 단정하게 담아냈습니다. Tauri v2 기술을 사용하여 Electron 기반의 무거운 앱들과 달리 시스템 리소스를 거의 차지하지 않는 극도의 경량감을 자랑합니다.
