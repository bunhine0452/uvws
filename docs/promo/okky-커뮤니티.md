# 한국 개발자 커뮤니티용 (OKKY / 커리어리 / 디스코드 / 페북 그룹 등)

캐주얼한 자기소개 톤. 그대로 복붙하거나 채널 분위기에 맞게 다듬어서 쓰세요.
AI 관련 커뮤니티(ComfyUI/SD 한국 디스코드·카톡방 등)에도 잘 맞습니다.
이 원고는 v0.5(uv 자동 설치) 출시 후를 전제로 합니다.

---

제목: ComfyUI랑 파이썬 앱 여러 개 띄우는 게 귀찮아서 데스크탑 런처를 만들었어요 (uvws)

---

요즘 로컬에서 ComfyUI나 파이썬으로 이것저것 돌릴 일이 많은데, 프로젝트마다 터미널 열어서 cd 하고 venv 켜고 실행하는 게 반복되니까 슬슬 지치더라고요. 하나 띄워놓고 다른 폴더에서 또 뭐 돌리면 터미널 탭이 정신없어지고요.

그래서 폴더 등록해두면 클릭 한 번으로 실행되는 데스크탑 앱을 만들었습니다. 이름은 uvws예요.

쓰면서 제일 편한 건 이런 것들이에요:

- 등록한 프로젝트 클릭 → 바로 실행. 내부적으로 uv run으로 돌려서 venv 신경 안 써도 되고, 없으면 알아서 만들어 줍니다. (uv 자체가 안 깔려 있어도 앱이 설치까지 해줘요.)
- 로그가 앱 안 터미널로 실시간으로 나오고, localhost 주소 뜨면 브라우저 열기 버튼이 자동 생성.
- "Kill Port" 버튼. 포트 충돌마다 lsof -i 치던 거 이걸로 끝. 해당 포트 프로세스 강제 종료하고 진짜 비워졌는지까지 확인해요.
- 여러 개 동시에 띄울 수 있고, 앱 끄면 돌던 서버 다 같이 정리됩니다.
- 공유 + QR: 실행 중인 로컬 서버를 공개 링크 + QR로 한 번에 노출해서, 폰으로 QR 찍어 바로 접속할 수 있어요. (ComfyUI 결과 폰으로 보여줄 때 편함)
- 그 외에 프로젝트별 CPU/메모리 실시간 표시, 구버전 패키지 업그레이드, Git 연동(브랜치/pull/push)도 있습니다.

Tauri로 만들어서 맥(M1~M4 / 인텔), 윈도우 다 됩니다.

아직 개인 사이드 프로젝트라 맥 공증은 못 받았고(첫 실행 보안 경고 우회법은 README에 있어요), 부족한 점 많을 텐데 써보고 피드백 주시면 정말 감사하겠습니다.
