# 한국 개발자 커뮤니티용 (OKKY / 커리어리 / 디스코드 / 페북 그룹 등)

캐주얼한 자기소개 톤. 그대로 복붙하거나 채널 분위기에 맞게 다듬어서 쓰세요.

---

제목: 파이썬 프로젝트 여러 개 돌리는 게 귀찮아서 데스크탑 앱 하나 만들었어요 (uvws)

---

요즘 로컬에서 파이썬으로 이것저것 돌릴 일이 많은데, 프로젝트마다 터미널 열어서 cd 하고 venv 켜고 실행하는 게 반복되니까 슬슬 지치더라고요. ComfyUI 같은 거 하나 띄워놓고 다른 폴더에서 또 뭐 돌리고 하면 터미널 탭이 정신없어지고요.

그래서 그냥 폴더 등록해두면 클릭 한 번으로 실행되는 앱을 만들었습니다. 이름은 uvws고요.

쓰면서 제일 편한 건 이런 것들이에요:

- 등록한 프로젝트 클릭 → 바로 실행. 내부적으로 uv run으로 돌려서 venv 신경 안 써도 되고, 없으면 알아서 만들어 줍니다.
- 로그가 앱 안에 터미널로 실시간으로 나오고, localhost 주소 뜨면 브라우저 열기 버튼이 자동으로 생겨요.
- "Kill Port" 버튼. 포트 충돌 날 때마다 lsof -i 치던 거 이걸로 끝냅니다. 해당 포트 프로세스 강제 종료하고 진짜 비워졌는지까지 확인해요.
- 여러 개 동시에 띄울 수 있고, 앱 끄면 돌던 서버 다 같이 정리됩니다.
- Git 연동도 있어서 브랜치 상태 보고 pull/push 버튼으로 처리 가능.

Tauri로 만들어서 맥(M1~M4 / 인텔), 윈도우 다 됩니다. 다만 uv는 미리 설치돼 있어야 해요.

아직 개인 사이드 프로젝트라 맥 공증은 못 받았고(첫 실행 때 보안 경고 우회 방법은 README에 있습니다), 부족한 점 많을 텐데 써보고 피드백 주시면 정말 감사하겠습니다.
