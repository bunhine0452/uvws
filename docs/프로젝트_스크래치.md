## 구상
1. MAC 용 앱으로 만들 생각. 그러나 tauri 를 이용한다면 윈도우 배포도 가능한지? 앱으로 배포하여 로컬에서 AI 프로젝트들을 쉽게 실행하는 앱을 만들거임. 배포도 할거임
2. 이 앱은 가볍고 쉬워야됨. 복잡한 기능은 필요없음. 그냥 git clone 하면 uv 환경 만들고 의존성 설치한다음에 main.py 또는 app.py 실행해주면 됨. 주로 로컬에서 돌아가는 Ai 프로젝트들이 이런 방식이지 않나?
3. UI 도 직관적이면 좋겠음. 너무 복잡하거나 화려한 UI를 바라는게 아님. 그냥 실용적이고 UX가 편리함에 최대 중점을 맞추기를 바람.
4. 터미널은 꼭 있어여함. 백엔드 실행이랄지 main.py 또는 app.py 실행 후 터미널에 로그를 보고싶음.

5. 어떤 포트에 열려있고 어떤 프로젝트인지 쉽게 알 수 있으면 좋겠음.
