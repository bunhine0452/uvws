## 제목

uvws – uv 기반 파이썬 프로젝트를 데스크탑 앱에서 클릭 한 번으로 실행

## 본문 (내용 칸에 붙여넣기)

로컬에서 파이썬 프로젝트를 여러 개 돌리다 보니 매번 터미널 열고 cd 하고 venv 켜고 python 돌리는 게 은근 일이라 만들어 봤습니다.

ComfyUI 같은 거 하나 띄우고, 다른 폴더에서 또 뭐 돌리고 하다 보면 터미널 탭이 금방 지저분해지더라고요. 포트 충돌 나면 lsof 쳐서 kill 하는 것도 매번 반복이고.

- 폴더 등록해두면 그다음부터 클릭 한 번으로 실행 (내부적으로 `uv run`으로 감싸서 venv 없어도 돌아가고, 없으면 첫 실행 때 자동으로 만들어줌)
- 각 프로젝트 로그가 앱 안 터미널(xterm.js)로 실시간으로 뜨고, 로그에 `localhost:포트` 나오면 브라우저 열기 버튼이 자동으로 생김
- 포트 강제 종료 버튼 — 그 포트 잡고 있는 프로세스 SIGKILL 때리고 진짜 비워질 때까지 확인함
- 여러 프로젝트 동시에 띄울 수 있고, 앱 끄면 돌던 서버들 같이 정리됨
- requirements.txt / pyproject.toml 있는 폴더 가져오면 파이썬 버전 골라서 의존성 설치까지 한 번에
- Git 저장소면 브랜치/변경사항 보여주고 fetch·pull·push 버튼

Tauri 2 + React로 만들었고 macOS(Apple Silicon/Intel), Windows 빌드 올려놨습니다. uv는 미리 깔려 있어야 해요.

아직 공증은 안 받아서 맥에서 첫 실행 때 보안 경고 뜨는데 README에 우회 방법 적어놨습니다. 피드백/이슈 환영합니다.
